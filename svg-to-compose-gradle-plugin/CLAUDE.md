# svg-to-compose-gradle-plugin

CRITICAL: MUST read and follow `AGENTS.md` in this directory for module-specific architecture, key files, and conventions.

This is the Gradle plugin that automates SVG/AVG to Compose conversion during builds.

```bash
./gradlew :svg-to-compose-gradle-plugin:build
./gradlew :svg-to-compose-gradle-plugin:test
./gradlew :svg-to-compose-gradle-plugin:detekt
```
