# Module svg-to-compose-gradle-plugin

A Gradle plugin that automates the conversion of SVG and Android Vector Drawable (AVD/XML)
files into Jetpack Compose `ImageVector` code as part of your build process.

Integrates with the `svg-to-compose` core library and supports incremental builds,
icon caching, and configurable parsing options.

See the [README](https://github.com/rafaeltonholo/svg-to-compose/tree/main/svg-to-compose-gradle-plugin)
for full documentation, DSL configuration, and setup instructions.
