# Module svg-to-compose

A Kotlin Multiplatform library that converts SVG or Android Vector Drawable (AVD/XML) files
into Android Jetpack Compose `ImageVector` code.

Supports JVM, macOS (x64/arm64), Linux (x64), and Windows (x64) targets.

See the [README](https://github.com/rafaeltonholo/svg-to-compose/tree/main/svg-to-compose)
for full documentation, installation instructions, and usage examples.
